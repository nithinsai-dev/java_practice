# 🧮 Basic Calculators (Java)

This folder contains a collection of **basic calculator programs implemented in Java**.  
These calculators are built to strengthen **core Java fundamentals**, logical thinking, and real-world problem solving.

---

## 📌 Calculators Included

- 📈 **Compound Interest Calculator**  
  Computes compound interest using standard financial formulas.

- **Language:** Java
- **Concepts Covered:**
    - Variables & Data Types
    - Conditional Statements
    - Loops
    - Methods
    - User Input (`Scanner`)
