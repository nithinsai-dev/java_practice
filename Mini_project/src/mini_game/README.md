# 🎮 Mini Games (Java)

This folder contains a collection of **console-based mini games developed using Java**.  
These games are created to practice **core Java concepts** while building fun, interactive programs.

---

## 🕹 Games Included

- ✊✋✌ **Rock Paper Scissors**  
  A simple game where the user plays against the computer using random choices.

- ❓ **Quiz Game**  
  A question-based game that evaluates the user's knowledge and displays the final score.

- 🎯 **Number Guessing Game**  
  The player guesses a randomly generated number within a limited number of attempts.

*(More games will be added as learning progresses.)*

---

## 🛠 Technologies Used

- **Language:** Java
- **Core Concepts Practiced:**
    - Conditional Statements
    - Loops
    - Methods
    - Random Number Generation
    - User Input using `Scanner`

- Learn Java through **game-based practice**
- Improve **logical thinking and problem-solving**
- Understand **control flow and user interaction**
- Prepare for **OOPS and DSA concepts**
